glyphs¶

A collection of dialogs to do things to the selected glyphs of one or more fonts.

class hTools2.dialogs.glyphs.copyMarginsDialog¶

A dialog to copy margins from selected glyphs in one font to the same glyphs in another font.

class hTools2.dialogs.glyphs.copyWidthsDialog¶

A dialog to copy the advance width of selected glyphs in one font to the same glyphs in another font.

class hTools2.dialogs.glyphs.copyToMaskDialog¶

A dialog to transfer the foreground layer of the selcted glyphs in the current font to the mask layer of the same glyphs of another font.

class hTools2.dialogs.glyphs.changeSuffixDialog¶

A dialog to change the suffix of the selected glyphs.

class hTools2.dialogs.glyphs.moveAnchorsDialog¶

A dialog to move the anchors in the selected glyphs of the current font.

class hTools2.dialogs.glyphs.moveGlyphsDialog¶

A dialog to move the selected glyphs in a font.

class hTools2.dialogs.glyphs.maskDialog¶

A dialog to transfer glyphs to and from the mask layer.

class hTools2.dialogs.glyphs.copyToLayerDialog¶

A dialog to copy the foreground layer in the selected glyphs to another layer.

class hTools2.dialogs.glyphs.copyPasteGlyphDialog¶

A dialog to copy and paste glyphs, with a few special options.

class hTools2.dialogs.glyphs.mirrorGlyphsDialog¶

A dialog to mirror the selected glyphs in the current font.

class hTools2.dialogs.glyphs.glyphActionsDialog¶

A dialog to apply actions to one or more layers in the selected glyphs.

class hTools2.dialogs.glyphs.interpolateGlyphsDialog¶

A dialog to interpolate the selected glyphs in one font with the same glyphs in another font into a third font.

class hTools2.dialogs.glyphs.paintGlyphsDialog¶

A dialog to apply a color to the selected glyph boxes, and to select glyphs by color.

class hTools2.dialogs.glyphs.rasterizeGlyphDialog¶

A dialog to rasterize the selected glyphs of the current font with element components.

class hTools2.dialogs.glyphs.renameAnchorsDialog¶

A dialog to rename the anchors in the selected glyphs of the current font.

class hTools2.dialogs.glyphs.roundToGridDialog¶

A dialog to round features of the selected glyphs to a grid.

class hTools2.dialogs.glyphs.shiftPointsDialog¶

A dialog to select and shift points in the selected glyphs in a font.

class hTools2.dialogs.glyphs.scaleGlyphsDialog¶

A dialog to scale the selected glyphs in a font.

class hTools2.dialogs.glyphs.skewGlyphsDialog¶

A dialog to skew the selected glyphs in a font.

class hTools2.dialogs.glyphs.slideGlyphsDialog¶

A dialog to slide the selected glyphs vertically and/or horizontally.

class hTools2.dialogs.glyphs.setWidthDialog¶

A dialog to set the advance width of the selected glyphs.

class hTools2.dialogs.glyphs.setMarginsDialog¶

A dialog to set the left/right side-bearings of the selected glyphs in the current font.

class hTools2.dialogs.glyphs.transferAnchorsDialog¶

A dialog to transfer anchors from selected glyphs in one font to the same glyphs in another font.
